Node binary websocket streaming made easy


## ダウンロード

サーバ

```console
$ npm install binaryjs
```
またはGitから
```console
$ git clone git://github.com/binaryjs/binaryjs.git
$ cd binaryjs 
$ npm install -g
```

クライアント

```html
<script src="http://cdn.binaryjs.com/0/binary.js"></script>
```
